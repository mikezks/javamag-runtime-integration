# Micro Frontend Architecture: Multi-Version / Multi-Framework

## Setup

- Angular 18: Shell
  
    ```bash
    cd angular-18
    npm install
    npx ng serve shell
    ```

- Angular 18: MFE Booking

    ```bash
    cd angular-18
    npx ng serve booking
    ```

- Angular 17: Boarding
  
    ```bash
    cd angular-17
    npm install
    npx ng serve boarding
    ```

- React 18: Checkin

    ```bash
    cd react-18
    npm install
    npm run dev
    ```
